Aprende de HTML y CSS

Fundamentos HTML:

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar páginas web, utilizando etiquetas para organizar contenido como texto, imágenes y enlaces. Se basa en elementos con etiquetas de apertura y cierre, creando una estructura jerárquica que los navegadores interpretan para visualizar el sitio

Fundamentos CSS:

CSS (Cascading Style Sheets) es el lenguaje utilizado para definir la presentación visual de documentos estructurados como HTML. Permite separar el contenido de la estructura (HTML) de su diseño (CSS), controlando colores, fuentes, diseño (layout), responsividad y animaciones. Funciona mediante reglas de selectores y propiedades que el navegador interpreta.

Box_Model

El modelo de caja (box model) en HTML/CSS es el concepto fundamental donde cada elemento es tratado como un rectángulo. Se compone de cuatro capas: Contenido (texto/imágenes), Padding (relleno interno), Border (borde) y Margin (espacio exterior). Estas capas determinan cómo se organizan, espacian y calculan las dimensiones de los elementos en la página.

Selectores

Los selectores en HTML/CSS permiten identificar y seleccionar elementos específicos de una página para aplicarles estilos o funcionalidades, siendo los fundamentales el selector de etiqueta (ej. p), de clase (.clase) y de ID (#id). Estos patrones definen qué elementos HTML son afectados por las reglas CSS.

Propiedades

Las propiedades en HTML, mejor conocidas como atributos, son palabras especiales añadidas dentro de la etiqueta de apertura para configurar el comportamiento o apariencia de un elemento. Se estructuran como nombre="valor" y son esenciales para definir aspectos como enlaces (href), fuentes de imagen (src), identificadores (id) o estilos (style).
