Introducción básica a HTML y CSS

HTML

Html es el lenguaje en el que se construye la base de la página web, ubicando titulos, textos, listas, links, etc. de forma ordenada.

CSS

CSS es la parte de la construcción de la página que se encarga del diseño, colores, tamaño de letra, fuente de la letra, efectos para que el visitante interactue con la página
